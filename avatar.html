SprinkleUI: Avatars

Avatars are found throughout different designs and can be used just about anywhere, from tables to dialog menus, from user info to status menu and so much more!

Image Avatars

Image avatars can be very useful in displaying user information, a list of friends etc. Since they're in such wide use already, finding them isn't that difficult a task. 

 
<!-- HTML code: -->

<div class="items">
    <img
        class="avatar-rg"
        src="./assets/avatarLogo.svg"
        alt="avatar-logo-small"
    />
    <img
        class="avatar-rg"
        src="./assets/avatar-2.svg"
        alt="avatar-logo-small"
    />
    <img
        class="avatar-rg"
        src="./assets/avatar-3.svg"
        alt="avatar-logo-small"
    />
    </div>

 
/* CSS code: */

.items {
padding: 0rem 0rem;
display: flex;
/* justify-content: space-around; */
}
.avatar-rg {
width: 3rem;
height: 3rem;
padding: 0rem 1rem;
}

Avatar sizes

Depending on the need or the design brief given, one may require avatars to be of a variety of sizes. Following are avatars in small, regular and large sizes respectively: 

 
<!-- HTML code: -->

<div class="items">

    <img class="avatar-sm" src="./assets/avatarLogo.svg" alt="avatar-logo-small" />

    <img class="avatar-rg" src="./assets/avatarLogo.svg" alt="avatar-log-regular" />
    
    <img class="avatar-lg" src="./assets/avatarLogo.svg" alt="avatar-logo-large" />

</div>

 
    /* CSS code: */
    
    .items {
    padding: 0rem 0rem;
    display: flex;
    /* justify-content: space-around; */
    
    .avatar-rg {
        width: 3rem;
        height: 3rem;
        padding: 0rem 1rem;
    }
    
    .avatar-sm {
        width: 2rem;
        height: 2rem;
    
        padding: 0rem 1rem;
    }
    
    .avatar-lg {
        width: 4rem;
        height: 4rem;
    
        padding: 0rem 1rem;
    }

Avatar status

Avatars with status badges can be used to denote a variety of things from busy to available, from offline to favourite etc can all be done using status badges. Following are some examples: 

 
<!-- HTML code: -->

<div class="items">

<div class="status">
<img class="avatar-rg" src="./assets/avatarLogo.svg" alt="avatar-logo-small" />
<img class="status-dot" src="./assets/red-dot.png" alt="" />
</div>

<div class="status">
<img class="avatar-rg" src="./assets/avatarLogo.svg" alt="avatar-logo-small" />
<img class="status-dot" src="./assets/green-dot.png" alt="" />
</div>

<div class="status">
<img class="avatar-rg" src="./assets/avatarLogo.svg" alt="avatar-logo-small" />
<img class="status-dot" src="./assets/orange-circle-96.png" alt="" />
</div>

</div>

 
    /* CSS code: */
    
    .items {
    padding: 0rem 0rem;
    display: flex;
    /* justify-content: space-around; */
    
    .status {
        position: relative;
    }
    
    .status-dot {
        height: 1rem;
        width: 1rem;
        position: absolute;
        bottom: 0.3rem;
        right: 1rem;
        border-radius: 50%;
        border: 3px solid white;
    }
    .avatar-rg {
        width: 3rem;
        height: 3rem;
        padding: 0rem 1rem;
    }
